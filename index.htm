2.创建网站项目

 

好了，上边的页面先停一下。先去激活一下邮件链接。

建议手机上下载“QQ邮箱”，接收邮件比较方便。为防止掺入个人手机信息，这里我用模拟器完成邮件验证（点击邮件中的https开头的蓝色链接）：

点击链接后，跳转到登录界面，可以登录试试。然后就激活了，激活后允许你在Github创建存储库：

回到Github这边。编程经验这个选项随意。主要是下边这个“您打算将GitHub用于什么？”（What do you plan to use GitHub for?），务必选择“使用GitHub Pages创建一个网站”（Creating a website with GitHub Pages）！！！

注：忘了选择“使用GitHub Pages创建一个网站”的话，待会创建存储库后去settings里手动开启一下。

点击“提交”，就可以啦。

点击网站右上角头像旁边的“+”，选择第一项“新存储库”（New repository），然后将会显示以下内容：

上图中，仓库名称为“你的用户名”.github.io。选择“公开”（Public），底部勾选“使用自述文件初始化此存储库”（Initialize this repository with a README）。然后点击底部绿色的“创建仓库”即可。

3.上传文件

 

上图中，很明显可以看到“上传文件”（Upload files）这四个字。这是上传文件的界面：

我个人并不习惯把文件拖拽进来。这里看你个人喜好。上传文件的进度条走完了之后，点击左下角绿色的“提交变更”。

重点说一下创建文件夹。上上个图，点击“上传文件”左边的“建立新档案”（Create new file）。

上图中，先关闭网页翻译，防止出现小写英文字母变成大写的尴尬。

不支持出现空白文件夹，所以我教你咋搞嗷。

输入文件夹名，你想要用的文件夹名字。输入文件夹名字后，再输入英文模式下的 “/”即可。随意新增一个文件，以及键入一些内容。注意编辑完之后，滚动到下方找到绿色确定按钮。即可新建一个文件夹。

4.比赛防作弊机制

 

在Github项目根目录下新建TXT文本，文本名统一命名为website.txt，文本内容为：你的院系+你的班级+你的姓名，如：曙光大数据学院 计BG183班 李易壮。

禁止抄袭他人的项目内容。我们会为大家提供一份网站模板，其他部分如若发现有雷同的情况（即两份或多份作品存在大量相同的HTML源代码），将根据项目文件上传&修改时间作为判断依据：

文件最后修改时间距离比赛截止时间最久远的，视为原创作品，其他的均视为抄袭。

注：大家不要在比赛群说出来你的网站链接是多少，更不要告诉其他人！一般是没人知道你网站地址的，你不把网址说出去，也就不会发生作品被抄袭的情况。

投票开始后，禁止再修改Github项目文件。否则视为自动弃权。

5.补充说明

 

本比赛指定使用Github平台进行网站免费托管，大家在做网站时不要加入个人敏感信息、用户登录及后台管理等成分。

上边手册内容所讲到的，或者其他细节操作，大家有什么不懂的地方，可以私聊问群管理员。

网站主页文件名一般默认为index.html，基础操作都不明白的，推荐你去菜鸟教程（点击即可跳转）学习一下HTML。

最好是自己写个404页面，这样网站比较完整。这些都会成为你作品的加分项！

提交作品时，须填写作品表（在比赛群群文件）并将此表添加到邮件附件，邮件内容为你网站的链接即可（如https://website2019.github.io），然后发送到203602445@qq.com（项目部部长郭秀方邮箱）。

本次比赛时间较短，如若有准备不周的地方，还望大家多多包涵。
